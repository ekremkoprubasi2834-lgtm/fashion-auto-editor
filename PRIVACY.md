# Privacy Policy

Last updated: June 18, 2026

Fashion Auto Editor is a local video production tool used to organize visual assets for personal video editing workflows.

## Information We Access

When connected to Pinterest, the tool may access the user’s Pinterest boards and pins that the user authorizes through Pinterest OAuth.

The tool may read:

* Board names and board IDs
* Pin titles and descriptions
* Pin media URLs
* Pin links and related metadata

## How We Use This Information

The information is used only to organize visual assets into local project folders for the user’s own video editing workflow.

The tool does not sell, share, or publish user data.

## Storage

Pinterest credentials and access tokens are stored locally by the user in their private `.env` file.

Local `.env` files are not committed to the repository.

Downloaded or organized assets are stored locally on the user’s computer.

## Third-Party Services

This tool may connect to Pinterest through the official Pinterest API and OAuth flow.

Pinterest’s own terms and privacy policies apply to Pinterest data and accounts.

## Data Deletion

Users can delete local downloaded assets, generated files, and tokens at any time from their computer.

Users can also revoke app access from their Pinterest account settings.

## Contact

For questions about this project, contact the repository owner.
